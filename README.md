# KFCM VFR Reporting Points

VFR feeder fixes and inside fixes for Flying Cloud Airport (KFCM), available as a ForeFlight content pack and Garmin GTN user waypoints.

The [latest release](https://github.com/nicknystrom/kfcm-vfr-reporting-points/releases/latest) includes:
- **foreflight-kfcm-vfr-waypoints.zip** — ForeFlight content pack
- **user.wpt** — Garmin GTN user waypoint file

## Installing in ForeFlight (iOS)

### One-tap install

Open this link on your iOS device with ForeFlight installed:

[Install KFCM VFR Waypoints in ForeFlight](https://foreflight.com/content?downloadURL=https://github.com/nicknystrom/kfcm-vfr-reporting-points/releases/latest/download/foreflight-kfcm-vfr-waypoints.zip)

Then enable the **KFCM VFR Waypoints** layer in Maps > Layers.

### Manual install

1. On your iOS device, open Safari and navigate to the [latest release](https://github.com/nicknystrom/kfcm-vfr-reporting-points/releases/latest).
2. Tap the **foreflight-kfcm-vfr-waypoints.zip** link to download it. Wait for the download to finish.
3. Open the **Files** app. The zip file should be visible in your Downloads. Long press on it, then tap **Share...**.
4. Select **ForeFlight** from the Share menu.
5. ForeFlight will open and display an **Import ZIP File** dialog.
6. Tap **Content Pack**. The import is now complete.
7. In ForeFlight, go to the **Maps** page, open the **Layers** menu, and enable **KFCM VFR Waypoints** at the bottom of the list.
8. You should see cyan feeder fixes and magenta inside fixes surrounding KFCM.

You can tap on the waypoints to add them to your flight plan, or use them as a visual reference combined with own-ship display when flying the KFCM VFR arrival procedures.

## Installing on Garmin GTN

1. Download **user.wpt** from the [latest release](https://github.com/nicknystrom/kfcm-vfr-reporting-points/releases/latest).
2. Copy the file to a blank SD card (8 GB or smaller).
3. Insert the SD card into the GTN's SD card slot.
4. Power up the GTN.
5. Navigate to **Waypoint Info** > **Import Waypoints** > **OK**.
6. Wait for confirmation, then power down.
7. Remove the SD card and re-insert your supplemental database card.

## Versioning

Releases use `<2-digit year>.<2-digit month>` versioning (e.g. `26.03` for March 2026). If multiple releases are needed in the same month, append a patch number (e.g. `26.03.1`).
